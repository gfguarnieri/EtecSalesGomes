Inputs

Exemplo 1:

Campo de Texto simples com placeholder:
Sobre os atributos do Input:

  • type: Define o tipo da entrada de dados.
    • text
    • date
    • email
    • file
    • number
  • placeholder: Apresenta um texto temporário no campo de texto informando o valor esperado no campo.
  • maxlength: Quantidade máxima de caracteres no campo
  • minlength: Quantidade mínima de caracteres no campo
  • autocomplete: Habilita (on) ou desabilita (off) sugestões no campo

Sobre o atributos do Label:
  • for: Relaciona o label ao input (precisa ter como valor o ID do input)

Exemplo 2:

Campo Numérico:
Sobre os atributos do Input do tipo numérico:

  • min: Define o valor mínimo permitido
  • max: Define o valor máximo permitido
  • value: Define um valor inicial para o campo
